# Roadmap

## Up next

- **Profiles** - named configurations (shell, font, theme, env vars) assignable per tab
- **Session restore** - reopen tabs, panes, and working directories on restart (WIP)
- **SSH connection manager** - saved hosts, key management, one-click connect as a tab
- **Inline images** - render images in terminal output (iTerm2 protocol, Sixel)
- **Scrollback search** - search the live terminal during command execution, not just completed blocks
- **Startup commands** - configurable commands that run when a new session opens

## Future

- **Plugin system** - extend Bolan with custom scripts and UI
- **Windows support** - Flutter supports Windows desktop, needs PTY and shell integration porting
- **Multiple snippet collections** - organize saved commands into groups
